Overview

This class provides information about XML SyntaxErrors. These exceptions are typically stored on Nokogiri::XML::Document#errors.

def to_s
  message = super.chomp
  [location_to_s, level_to_s, message].
    compact.join(": ").
    force_encoding(message.encoding)
end
